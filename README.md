# zig-3ds

![](screenshot.webp)

## Getting started

- [zig](https://ziglang.org/download/)
- [devkitPro](https://devkitpro.org/wiki/Getting_Started)

```
pacman -S 3ds-dev
git clone https://github.com/zig-homebrew/zig-3ds
cd zig-3ds/
zig build # then run zig-out/zig-3ds.3dsx with citra
```
